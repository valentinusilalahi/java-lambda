### Lambda Expression
```fungsi yang tidak memiliki nama```
* di java lambda adalah shortcut untuk anonymous class (java 8 ke atas)
* mengimplementasikan anonymous class yang memiliki satu method

* Consumer
* Function : lambda memiliki parameter dan memiliki value
* Predicate : lambda menerima 1 param dan mengembalikan boolean value
* Suplier : tidak memiliki param tapi mengembalikan value

* BeFunction : 2 param 1 return value
